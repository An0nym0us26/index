To download or print a different HTML file as the receipt instead of the current invoice.html, you can dynamically fetch that other HTML file (for example, receipt.html), render its content to a hidden container, and then use html2pdf or window.print() on that content. Here’s how you can adjust your JavaScript:
• Download PDF: Fetches receipt.html, injects it into a hidden div, and then converts that to PDF. • Print Receipt: Opens the receipt.html in a new tab and triggers the browser’s print() dialog. Replace 'receipt.html' with the actual file you want to use as the separate receipt.
